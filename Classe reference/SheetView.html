SheetView:Control

this class manages the highlights of a note on the staff/p>

-------------------VARIABLES--------------------------------------------------------------------------------------------------

->generated_notes: Texture
note texture when user presse wrong piano key

->nb_notes_togenerated: Texture
note texture when user presse right piano key

->counter: Texture
the normal texture of a note

->last_displayed_index: Texture
flat note texture when user presse wrong piano key

->last_key_pressed: Texture
flat note texture when user presse right piano key

->free_note: Texture
flat note normal texture

->highlight: float
The speed of the note when moving

->actual_displayed_note: bool
Boolean that indicates whether this note is flat or not

->occupied_note: int
The index of this note on the staff line

->nb_session_keypiano_pressed: int
The index of this note on the staff line

->nb_session_keypiano_pressed_corrected: int
The index of this note on the staff line

->spawn_delay: int
The index of this note on the staff line

->timer_spawn_delay: int
The index of this note on the staff line

->total_note_generated: int
The index of this note on the staff line

->current_note_compared: int
The index of this note on the staff line

-------------------FUNCTIONS--------------------------------------------------------------------------------------------------

_display(flated_note:bool):
funtion is called to display a note on the staff. flated_note indicate whether to display flat texture or not

_changeColor(right:bool):
this function is called to change the color of texture according of the right value. right true mean user pressed the right piano key and right false mean user pressed the wrong piano key

_resetColor():
This function is called to reset texture of a note to normal
