ConfigMenu:Node2D

this class manages all behaviours elements on the ConfigMenu scene

-------------------VARIABLES--------------------------------------------------------------------------------------------------

->btn_scale_anim: Vector2D
The maximum value of the scale of the buttons when pressed

->btn_anim_time: float
the animation time of the button to go from normal scale to max scale and vice vesa

-------------------FUNCTIONS--------------------------------------------------------------------------------------------------

_on_btn_pressed():
This function is called when the user releases the back button

_on_btnBack_button_down():
This function is called when the user presses the back button

_on_btnAdd_button_down():
This function is called when the user presses the increase tempo button

_on_btnRemove_button_down():
This function is called when the user presses the decrease tempo button

_on_btnRemove_pressed():
This function is called when the user releases the decrease tempo button

_on_btnAdd_pressed():
This function is called when the user releases the increase button
