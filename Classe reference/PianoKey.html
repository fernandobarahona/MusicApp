PianoKey:Node

this class handle a piano key behaviours

-------------------VARIABLES--------------------------------------------------------------------------------------------------

->key_note: [GameManager.Note]
Note played by this piano key

->flated: bool
Boolean indicating whether this piano key play a flat note or not

-------------------FUNCTIONS--------------------------------------------------------------------------------------------------

_pressed():
This function is called when user releases the piano key nbNote

_button_down():
This function is used to generate music note according to the number specified in the variable nbNote
