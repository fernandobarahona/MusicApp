NotesGenerator:Node

this class contains differents behaviours needed to generating list of notes to display later on the sheetView

-------------------VARIABLES--------------------------------------------------------------------------------------------------

->generated_notes: [GameManager.Note]
This variable contains the list of notes generated by the generator. these notes will be displayed on the music score.

->tempo: GameManager.Octave
The tempo of music

->possibleNotes: bool
The diferent basic notes to generate the final notes of a music

-------------------FUNCTIONS--------------------------------------------------------------------------------------------------

_generetate_notes(nbNote):
This function is used to generate music note according to the number specified in the variable nbNote
