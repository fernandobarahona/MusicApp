GameHUDLeft:Control

this class manages the percentage bar on game scene(top left corner)

-------------------VARIABLES--------------------------------------------------------------------------------------------------

->last_20_try: []
this list keeps the result(good and bad) of the user's evaluation after pressing the piano keys

->last_20_try_index: float
the current index out of 20 to evaluate the last 20 attempts of the user

->last_20_try_corrected: float
Number of good results over the user's last 20 tests

->last_20_try_size: float
