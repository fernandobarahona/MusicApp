SheetView:Control

this class manages all SheetView behaviours

-------------------VARIABLES--------------------------------------------------------------------------------------------------

->generated_notes: [Note]
list all notes generated to create a musical sheet

->nb_notes_togenerated: int
number of notes to generate to create a musical sheet

->last_displayed_index: int
index of last note displayed from the generated_notes

->last_key_pressed: Button
last button pressed on the piano

->highlight: SheetHighlight
temp variable to stock the latest instantiate highlight note

->actual_displayed_note: []
List of notes already displayed on the sheet.

->actual_displayed_highlight: []
List of notes highlight already displayed on the sheet.

->nb_session_keypiano_pressed: int
Number of piano key pressed since the session started

->nb_session_keypiano_pressed_corrected: int
Number of correct piano key pressed since the session started

->spawn_delay: float
time waiting time to create the next note on the SheetView

->timer_spawn_delay: float
a temp variable to count the waiting time before creating next note on the sheetView. next note is created when his value reach 0

->total_note_generated: int
Total note instantiated on the sheet since the music started

->current_note_compared: int
index of current sheet note to compare depending on what piano key the user pressed

-------------------FUNCTIONS--------------------------------------------------------------------------------------------------

_display_note(note:Note):
This function is called to displaye visually a note on the sheetView. these note come from the generated_notes list

_on_Piano_piano_key_pressed(note:Note, btn:Button):
This function responds to the piano_key_pressed event emitted when the user presses a piano key. it evaluate the key pressed according to the note displayed

_instantiate_note(note:Note):
This function is called to instantiate a highlight note on the staff depending on the note value(CDEFGAB...)
